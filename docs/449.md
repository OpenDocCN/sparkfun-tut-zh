# 仙女 LED 手镯

> 原文：<https://learn.sparkfun.com/tutorials/fairy-led-bracelet>

## 介绍

在这个项目中，我们将使用仙女 LED 灯和树脂浇铸的可充电 LiPo 电池制作一个可佩戴的手镯。

设计建造时间:30 分钟— 1 小时
树脂干燥时间:24 小时
