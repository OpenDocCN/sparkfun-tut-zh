# LED 水晶女神皇冠

> 原文：<https://learn.sparkfun.com/tutorials/led-crystal-goddess-crown>

## 介绍

设计和建造时间:2 天(使用 7 个模具)

在这个项目中，我们将使用一个简单的电路制作一个可佩戴的 LED 水晶皇冠，该电路由 7x [超高亮白色 LED](https://www.sparkfun.com/products/531)并联组成，由两个串联的纽扣电池供电。
